##### Cambiar estado de pedido

```js
// Request
axios.post(`${apiUrl}/change-state-pedido`, {
    pedido: 23,
    state: 1, // 1. entregado, 2. pendiente, 3. cancelado
});
```

```json
// Respuesta
{
    "status": 200,
    "response": "success",
    "message": "Actualizado."
}
```
